# Segregation-Dynamics-on-Lattices-and-Complex-Networks
This repository contains the code developed for an exam project in **Problem Solving in Physics** at the **University of Pavia**.

The project investigates segregation dynamics in the Schelling model, starting from spatial lattices and then extending the analysis to complex networks.

The repository contains two main notebooks:

* `01_schelling_lattice_dynamics.ipynb`: implementation of the Schelling model on square lattices, with visualisation of the dynamics and analysis of threshold-like behaviour.

* `02_schelling_complex_networks.ipynb`: extension of the model to complex networks, including different network topologies, segregation indices, rewiring experiments and noisy update dynamics.

The code is used to generate the figures and simulations discussed in the final presentation.
